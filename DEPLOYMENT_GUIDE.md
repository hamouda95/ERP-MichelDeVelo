# 🚀 DÉPLOIEMENT PROFESSIONNEL ERP MICHEL DE VÉLO

## Script Unique de Déploiement

### Windows (recommandé)
```bash
deploy.bat
```

### Linux/Mac
```bash
bash deploy.sh
```

## ✅ Ce que fait le script

1. **Backend** : Migrations + Redémarrage
2. **Frontend** : Build + Redémarrage  
3. **Database** : Synchronisation automatique
4. **Tests** : Vérification des services

## 🎯 Résultat

- Backend : http://localhost:8000
- Frontend : http://localhost:3001
- Database : ✅ Synchronisée

```bash
deploy.bat
```
